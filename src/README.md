# Source Code

This directory contains the implementation of the Moss engine.

## Internal Function Naming Conventions

**Functions**: `moss__xxx` (e.g., `moss__create_api_instance`, `moss__init_buffer_sharing_mode`)

**Structures**: `Moss__XXX` (e.g., `Moss__CreateSwapchainInfo`, `Moss__SelectSuitableMemoryTypeInfo`)

Internal functions and structures use double underscores (`__`) to distinguish them as internal implementation details, separate from the public API.
